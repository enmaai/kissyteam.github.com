.. function:: Editor.commands.foreColor

    | **foreColor(color)**
    | 对选区设置字体颜色插件

    :param String color: 颜色的 16 进制字符串。例如 '#ffffff'


.. note::

    调用命令方式

    .. code-block:: javascript

        editor.execCommand("foreColor",'#ffffffff'); // 设置选区字体颜色为白色
