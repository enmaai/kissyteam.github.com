dom.wrap()¶
void wrap ( selector,wrapperNode )
用 wrapperNode 分别包装符合 selector 的节点
Parameters:
 
  • selector (string|HTMLCollection|Array<HTMLElement>) – 字符串格式参见 KISSY selector
  • wrapperNode (HTMLElement) – 包装节点
 
<div class="container">
  <div class="inner">Hello</div>
  <div class="inner">Goodbye</div>
</div>

after

DOM.wrap(".inner",DOM.create('<div class="new" />'));

become

<div class="container">
  <div class="new">
    <div class="inner">Hello</div>
  </div>
  <div class="new">
    <div class="inner">Goodbye</div>
  </div>
</div>
