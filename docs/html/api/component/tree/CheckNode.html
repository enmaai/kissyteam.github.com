{Enum} - 当前节点的多选状态. 可取：

CheckNode.PARTIAL_CHECK¶

<static> - 代表节点的部分选中状态（部分子孙节点选中）

CheckNode.CHECK¶

<static> - 代表节点的全部选中状态（子孙节点和自身节点全部选中）

CheckNode.EMPTY¶

<static> - 代表节点的空状态（子孙节点和自身节点全部不选中）
