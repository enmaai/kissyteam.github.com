Configs Detail¶

Carousel 接口及配置项, 与 Switchable 相同, 其中以下配置项的默认值有所区别:

Carousel.config.aria¶

{Boolean} - 是否支持键盘以及 aria. 默认 false

Switchable.config.circular

{Boolean} - 默认为true, 是否循环切换.

Switchable.config.prevBtnCls¶

{String} - 默认为 ‘ks-switchable-prev-btn’, 前一个触发器的 cls.

Switchable.config.nextBtnCls¶

{String} - 默认为 ‘ks-switchable-next-btn’, 后一个触发器的 cls.

Switchable.config.disableBtnCls¶

{String} - 默认为 ‘ks-switchable-disable-btn’, 触发器不可用时的 cls.
