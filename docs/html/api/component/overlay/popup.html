Class Detail¶

class overlay.Popup¶
Popup (config)
xclass: ‘popup’
  • 继承自 Overlay , 包含其全部配置,属性,方法,事件.
Parameters:
 config (Object) – 配置项, 详细见下方 Configs Detail .

Note

使用前请加上初始样式

<style>
    .ks-overlay {
        position:absolute;
        left:-9999px;
        top:-9999px;
    }
</style>

Configs Detail¶

Popup.prototype.trigger¶

{String|HTMLElement|KISSY.Node} - 触点

Popup.prototype.triggerType¶

{String} - 可选, 默认为’click’, 触发类型, 可选’click’, ‘mouse’.

Popup.prototype.mouseDelay¶

{Number} - 可选, triggerType 为 mouse 时, Popup 显示的延迟时间, 默认为 100ms.

Popup.prototype.toggle¶

{Boolean} - 可选, triggerType 为 click 时, Popup 是否有toggle功能，默认为false，不开启
