    // use Menu.SubMenu
});

class menu.SubMenu¶
SubMenu (config)
xclass: ‘submenu’
  • 继承自 Item , 包含其全部配置,属性,方法,事件.
Parameters:
 config (Object) – 配置项，详细见下方 Configs Detail .

Note

从已有元素实例 Menu 对象时 , 元素节点标明类名 {prefixCls}submenu . prefixCls 为配置的类名前缀，默认为 ks- .

Configs Detail¶

SubMenu.config.menuDelay¶

{Number} - 可选，默认 0.15，单位秒。鼠标掠过后多长时间显示子菜单.

SubMenu.config.menu¶

{Menu} - 子菜单实例.

SubMenu.config.externalSubMenu¶

{Boolean} - 可选，默认 false . 销毁该菜单项是否顺带销毁子菜单.
