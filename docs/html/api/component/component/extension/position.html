extension.Position.config.zIndex¶

{Number} - 可选, 默认为 9999, 设置组件根节点的 z-index 值.

{Number} - 可选, 组件根节点相对于文档根节点的 x 坐标.

{Number} - 可选, 组件根节点相对于文档根节点的 y 坐标.
